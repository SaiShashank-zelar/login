# Login:

### Install Golang:-

* #apt update

* #wget -c https://dl.google.com/go/go1.14.2.linux-amd64.tar.gz -O - | sudo tar -xz -C /usr/local

* #export PATH=$PATH:/usr/local/go/bin

* #source ~/.profile

* #go version

* #mkdir /go,cd /go

* #mkdir src,cd src

*git clone https://github.com/zelar-soft-todoapp/login.git,cd login

* export GOPATH=/go
